# The Kenya Wildlife Service Website

To get started:

1. Clone the repository:


bash
git clone git@bitbucket.org:bitfactor/kws-landing-template.git

cd kws-landing-template

2. Install the dependencies:

bash

# Using npm

npm install

# Using Yarn

yarn

3. Start the development server:

bash

# Using npm

npm run serve

# Using Yarn

yarn serve
